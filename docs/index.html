GitHub アクセストークンの設定(初回のみ)

  1. GitHub のトークン作成ページ ↗ を開く
  2. Token name: 任意(例:「old-domain uploader」)
  3. Expiration: 好みで(90日推奨)
  4. Repository access → Only select repositories → junf66/old-domain を選択
  5. Repository permissions → Contents を Read and write に設定
  6. 下部の Generate token → 表示されたトークンをコピー
  7. そのトークンを下に貼り付け → 保存

トークンはこのブラウザの localStorage にのみ保存されます(サーバには送信されません)。
